# Library Management System

A full-stack Library Management System built using MERN Stack.

## Features

- Student Registration & Login
- Admin Login
- Add, Update and Delete Books
- Borrow Books
- Return Books
- Due Date Tracking
- Overdue Reminders
- Dashboard Statistics
- Search and Filter Books

## Tech Stack

### Frontend

- React
- React Router
- Axios
- CSS

### Backend

- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
